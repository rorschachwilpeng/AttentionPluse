# Web Demo (B线 - 评审体验)

网页版演示，用于让评委无需安装插件即可体验 AttentionPulse 的核心价值。

## 目录结构

```
web-demo/
├── src/                  # 源代码
│   ├── index.html        # 主页面
│   ├── styles/           # 样式文件
│   ├── scripts/          # JavaScript 逻辑
│   └── components/       # 组件（如需要）
├── assets/               # 静态资源
│   └── images/           # 图片资源
└── public/               # 公共文件
```
