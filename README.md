## UI 设计记录

### AttentionPulse Widget 外观设计

#### 整体外观
- **容器样式**：深灰色、圆角矩形背景
- **标题**：顶部显示 "AttentionPulse"（英文标题）
- **副标题**：中文副标题（如"注意力波动图"）
- **窗口控制**：左上角三个彩色圆点（红、黄、绿，类似 macOS 风格）

#### 核心视觉元素 - 波形图（Attention Pulse）
- **形态**：动态、持续波动的波形图（心电图/波谱隐喻）
- **颜色渐变**：水平渐变，从左到右：
  - 绿色 → 黄色 → 橙色 → 红色 → 紫色 → 蓝色
- **振幅变化**：波形振幅沿长度变化，反映信息密度
- **动画**：持续波动，低存在感

#### 类别指示器（Category Indicators）
- **位置**：波形图下方
- **样式**：四个彩色圆圈，每个对应一个内容类别
  - 🟢 绿色："技"（科技/技术）
  - 🔵 蓝色："家习"（学习）
  - 🔴 红色："极乐"（娱乐）
  - 🩷 粉色："生活"（生活）
- **功能**：显示当前内容类型与颜色的映射关系

#### 位置与布局
- **主要位置**：页面右侧固定位置（不干扰内容浏览）
- **可选位置**：左上角浮动 widget（较小尺寸）
- **主题适配**：深色主题，与小红书深色界面融合
- **非侵入性**：不修改原页面结构，不影响滚动与原生交互

#### 设计原则
- 低存在感：默认状态下几乎被忽略
- 持续存在：始终显示，无需用户操作
- 视觉反馈：通过颜色和振幅变化反映注意力状态
